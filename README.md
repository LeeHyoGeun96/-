# Color Blind Extension

색각이상자를 위한 Chrome 확장 프로그램입니다.

## 기능
- 이미지의 색상을 도형으로 변환
- 색각이상자가 구분하기 쉬운 형태로 표시
- 실시간 이미지 처리 및 변환

## 설치 방법
1. Chrome 확장 프로그램 페이지 열기
2. '압축해제된 확장 프로그램을 로드합니다' 클릭
3. 이 프로젝트 폴더 선택

## 사용 방법
1. 확장 프로그램 아이콘 클릭
2. 원하는 픽셀 사이즈 클릭

## 개발 환경
### 기술 스택
- Chrome Extension Manifest V3
- HTML5 Canvas API
- Paper.js (벡터 그래픽 라이브러리)
- JavaScript ES6+

### 주요 라이브러리
- Paper.js: 벡터 그래픽 처리
- Canvas API: 이미지 처리 및 픽셀 조작
